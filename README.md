# Educational AI Agent Dashboard

An interactive student dashboard featuring a comprehensive Educational AI Agent system that provides personalized learning experiences through advanced AI technologies.

## Features

- **Personalized Student Knowledge Graph**: Dynamic learning network for each student
- **Knowledge Tracing**: SQKT (Students' Question-based Knowledge Tracing) with MLFBK model
- **Cognitive Diagnosis**: GNN-based models with LLM cold-start solutions
- **Educational Agent**: RL-based recommendation system with specialized AI tools
- **Explainable AI**: Transparent decision-making for student agency and metacognition

## Technologies Used

- **Backend**: Flask, Python
- **AI/ML**: PyTorch, Transformers, scikit-learn, NetworkX
- **Visualization**: matplotlib, mpld3, plotly, seaborn
- **Frontend**: HTML, CSS, JavaScript, Bootstrap

## Installation

1. Install dependencies using uv:
```bash
uv sync
```

2. Activate the virtual environment:
```bash
source .venv/bin/activate
```

3. Run the application:
```bash
python app.py
```

## Usage

Navigate to `http://localhost:5003` to access the student dashboard.

## Components

- **Knowledge Tracing Dashboard**: SQKT analysis and knowledge state evolution
- **Cognitive Diagnosis Visualizations**: Personalized knowledge graphs and GNN message passing
- **RL Agent Decision Dashboard**: Recommendation system with AI tool integration
- **Explainable AI Interface**: Transparent AI reasoning and decision trees
